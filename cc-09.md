# JDBC CRUD Operations using AWS EC2 and RDS

## Objective

To perform CRUD (Create, Read, Update, Delete) operations on a MySQL database hosted on AWS RDS using a Java JDBC application running on an AWS EC2 instance.

---

## Architecture

```text
Java Application
       ↓
      EC2
       ↓ JDBC
      RDS MySQL
```

---

## Prerequisites

* AWS Account
* EC2 Instance (Ubuntu)
* RDS MySQL Database
* Java JDK 17+
* MySQL Connector/J JDBC Driver

---

## Step 1: Create an RDS MySQL Database

1. Open **AWS Console → RDS → Create Database**
2. Select:

   * Engine: MySQL
   * Template: Free Tier
3. Configure:

   * DB Identifier: `database-2`
   * Username: `admin`
   * Password: `<your-password>`
4. Create the database.
5. Note the RDS Endpoint.

Example:

```text
database-2.xxxxxxxxx.eu-north-1.rds.amazonaws.com
```

---

## Step 2: Create an EC2 Instance

1. Launch an Ubuntu EC2 instance.
2. Configure Security Group:

   * SSH (Port 22) from your IP.
3. Connect using SSH:

```bash
ssh -i key.pem ubuntu@<EC2-PUBLIC-IP>
```

---

## Step 3: Install Java

```bash
sudo apt update
sudo apt install openjdk-17-jdk -y

java -version
javac -version
```

---

## Step 4: Configure RDS Security Group

Add an inbound rule:

```text
Type: MySQL/Aurora
Port: 3306
Source: EC2 Security Group
```

This allows EC2 to connect to RDS.

---

## Step 5: Connect to RDS

Install MySQL Client:

```bash
sudo apt install mysql-client -y
```

Connect:

```bash
mysql -h <RDS-ENDPOINT> -u admin -p
```

Create database:

```sql
CREATE DATABASE employee_db;
USE employee_db;
```

Create table:

```sql
CREATE TABLE employees(
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100),
    country VARCHAR(50),
    salary DOUBLE
);
```

## Step 7: Create Java Program

Create the file:

```bash
nano JdbcCrudDemo.java
```

Paste the following code:

```java
import java.sql.*;

public class JdbcCrudDemo {

    static final String JDBC_URL =
            "jdbc:mysql://database-2.c5yimwcqgy8n.eu-north-1.rds.amazonaws.com:3306/employee_db";

    static final String JDBC_USER = "admin";
    static final String JDBC_PASS = "your-password";

    public static void main(String[] args) {

        try {
            Class.forName("com.mysql.cj.jdbc.Driver");

            Connection conn = DriverManager.getConnection(
                    JDBC_URL,
                    JDBC_USER,
                    JDBC_PASS
            );

            // CREATE
            String insertSql =
                    "INSERT INTO employees (name, email, country, salary) VALUES (?, ?, ?, ?)";

            PreparedStatement insertStmt = conn.prepareStatement(insertSql);

            insertStmt.setString(1, "Alice");
            insertStmt.setString(2, "alice@example.com");
            insertStmt.setString(3, "USA");
            insertStmt.setDouble(4, 50000);

            insertStmt.executeUpdate();

            System.out.println("\nInserted new employee.");

            // READ
            String selectSql = "SELECT * FROM employees";

            Statement selectStmt = conn.createStatement();
            ResultSet rs = selectStmt.executeQuery(selectSql);

            System.out.println("\nEmployee List After Insert:");

            while (rs.next()) {
                System.out.println(
                        rs.getInt("id") + ", " +
                        rs.getString("name") + ", " +
                        rs.getString("email") + ", " +
                        rs.getString("country") + ", " +
                        rs.getDouble("salary")
                );
            }

            // UPDATE
            String updateSql =
                    "UPDATE employees SET salary = ? WHERE name = ?";

            PreparedStatement updateStmt =
                    conn.prepareStatement(updateSql);

            updateStmt.setDouble(1, 60000);
            updateStmt.setString(2, "Alice");

            updateStmt.executeUpdate();

            System.out.println("\nUpdated salary for Alice.");

            rs = selectStmt.executeQuery(selectSql);

            System.out.println("\nEmployee List After Update:");

            while (rs.next()) {
                System.out.println(
                        rs.getInt("id") + ", " +
                        rs.getString("name") + ", " +
                        rs.getString("email") + ", " +
                        rs.getString("country") + ", " +
                        rs.getDouble("salary")
                );
            }

            // DELETE
            String deleteSql =
                    "DELETE FROM employees WHERE name = ?";

            PreparedStatement deleteStmt =
                    conn.prepareStatement(deleteSql);

            deleteStmt.setString(1, "Alice");

            deleteStmt.executeUpdate();

            System.out.println("\nDeleted employee Alice.");

            rs = selectStmt.executeQuery(selectSql);

            System.out.println("\nEmployee List After Delete:");

            boolean hasRows = false;

            while (rs.next()) {
                hasRows = true;

                System.out.println(
                        rs.getInt("id") + ", " +
                        rs.getString("name") + ", " +
                        rs.getString("email") + ", " +
                        rs.getString("country") + ", " +
                        rs.getDouble("salary")
                );
            }

            if (!hasRows) {
                System.out.println("No records found.");
            }

            rs.close();
            insertStmt.close();
            updateStmt.close();
            deleteStmt.close();
            selectStmt.close();
            conn.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

---

## Step 8: Compile the Program

```bash
javac -cp mysql-connector-j-9.3.0/mysql-connector-j-9.3.0.jar JdbcCrudDemo.java
```

---

## Step 9: Run the Program

```bash
java -cp .:mysql-connector-j-9.3.0/mysql-connector-j-9.3.0.jar JdbcCrudDemo
```
