# JDBC CRUD Operations using AWS EC2 and RDS

## Objective

To perform CRUD (Create, Read, Update, Delete) operations on a MySQL database hosted on AWS RDS using a Java JDBC application running on an AWS EC2 instance.

---

## Prerequisites

* AWS Account
* EC2 Instance (Ubuntu)
* RDS MySQL Database
* Java JDK 17+
* MySQL Connector/J JDBC Driver

---

## Step 1: Create an RDS MySQL Database

1. Open **AWS Console → RDS → Create Database**
2. Select:

   * Engine: MySQL
   * Template: Free Tier
3. Configure:

   * DB Identifier: `database-2`
   * Username: `admin`
   * Password: `<your-password>`
4. Enable Public Access (for testing).
5. Create the database.
6. Note the RDS Endpoint.

Example:

```text
database-2.xxxxxxxxx.eu-north-1.rds.amazonaws.com
```

---

## Step 2: Create an EC2 Instance

1. Launch an Ubuntu EC2 instance.
2. Configure Security Group:

   * SSH (Port 22) from your IP.
3. Connect using SSH:

```bash
ssh -i key.pem ubuntu@<EC2-PUBLIC-IP>
```

---

## Step 3: Install Java

```bash
sudo apt update
sudo apt install openjdk-17-jdk -y

java -version
javac -version
```

---

## Step 4: Configure RDS Security Group

Add an inbound rule:

```text
Type: MySQL/Aurora
Port: 3306
Source: EC2 Security Group
```

This allows EC2 to connect to RDS.

---

## Step 5: Connect to RDS

Install MySQL Client:

```bash
sudo apt install mysql-client -y
```

Connect:

```bash
mysql -h <RDS-ENDPOINT> -u admin -p
```

Create database:

```sql
CREATE DATABASE employee_db;
USE employee_db;
```

Create table:

```sql
CREATE TABLE employees(
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100),
    country VARCHAR(50),
    salary DOUBLE
);
```

---

## Step 6: Download JDBC Driver

```bash
wget https://dev.mysql.com/get/Downloads/Connector-J/mysql-connector-j-9.3.0.tar.gz
tar -xvf mysql-connector-j-9.3.0.tar.gz
```

Verify:

```bash
ls mysql-connector-j-*/mysql-connector-j-*.jar
```

---

## Step 7: Create Java Program

```bash
nano JdbcCrudDemo.java
```

Paste the JDBC CRUD code and update:

```java
static final String JDBC_URL =
"jdbc:mysql://<RDS-ENDPOINT>:3306/employee_db";

static final String JDBC_USER = "admin";
static final String JDBC_PASS = "<password>";
```

---

## Step 8: Compile the Program

```bash
javac -cp mysql-connector-j-9.3.0/mysql-connector-j-9.3.0.jar JdbcCrudDemo.java
```

---

## Step 9: Run the Program

```bash
java -cp .:mysql-connector-j-9.3.0/mysql-connector-j-9.3.0.jar JdbcCrudDemo
```

---

```text

Employee List After Insert:

Employee List After Update:

```

---

## Result

Successfully connected an AWS EC2-hosted Java application to an AWS RDS MySQL database using JDBC and performed CRUD operations.
